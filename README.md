Alaska Logbook
===============

Mobile data collection for Alaska fish surveys

Install
========
Requirements:
* Vagrant & VirtualBox
* git


```
git clone https://github.com/Ecotrust/ak-logbook.git
git clone https://github.com/Ecotrust/formhub.git
git clone https://github.com/Ecotrust/enketo.git

cd ak-logbook
vagrant up
fab dev init
```
